# lahhh-YTdl
Using this tool you can easily Download YouTube, Facebook, Instagram videos and audio using Termux.

## Usage:
### First Do Setup:
```bash
apt install wget -y

wget https://github.com/onlyabay/lahhh-YTdl/raw/main/setup.sh

chmod +x setup.sh

./setup.sh
```
